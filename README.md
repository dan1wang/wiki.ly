# Wiki.ly

_This is a fork of the MediaWiki's Parsoid_

A Node.js libary for converting Mediawiki wikitext to
a list of tokens.

* Example input: [gratis.txt](tests/gratis.txt)
* Example output: [gratis.tokens](tests/gratis.tokens)
